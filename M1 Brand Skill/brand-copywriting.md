# emilybot — M1 Copy Guide

A modular brand voice guide that equips anyone to write first-draft M1 copy that sounds like Emily wrote it.

---

## How to use this skill

**Step 1: Always load these three files first.**

Read them before writing a single word of copy:

- `copywriting/copywriting-philosophy.md` — M1's north star, who we're writing for, how to handle a brief
- `copywriting/copywriting-voice-tone.md` — energy, formality, writing principles, non-negotiables
- `copywriting/copywriting-compliance.md` — FINRA 2210 rules, what to never say, and the full coded disclosure library (General, Advisor, Invest, Margin, Cash Account, Savings, Personal Loans, Crypto, Owner's Rewards Card)

**Step 2: Load the channel file that matches the task.**

| Task | File |
|------|------|
| Marketing or transactional email | `copywriting/copywriting-channel-email.md` |
| Push notification or in-app message | `copywriting/copywriting-channel-push-inapp.md` |
| Web page or landing page | `copywriting/copywriting-channel-web.md` |
| Blog post | `copywriting/copywriting-channel-blog.md` |
| In-product / UX microcopy | `copywriting/copywriting-channel-ux-copy.md` |

If the task spans multiple channels (e.g., an email + push notification campaign), load all relevant channel files.

**Step 3: Write.**

Interpret the brief — don't regurgitate it. See `copywriting/copywriting-philosophy.md` for the "Working with briefs" section.

## Quick reference: The non-negotiables

Before submitting any copy, confirm:

- [ ] Biggest gun first — most compelling point leads every section
- [ ] Sentence case on all headlines, subheads, subject lines (never title case)
- [ ] Emails have a headline, under 10 words, favoring action verbs/directive phrasing (subheads are optional)
- [ ] No em-dashes — recast with a period, comma, or colon
- [ ] Benefit over feature ("does the work for you" not "automates investing")
- [ ] No guarantees or promises ("can help" not "will help")
- [ ] No telling readers what they SHOULD do
- [ ] For web, email, or blog copy — correct coded disclosure(s) from `copywriting/copywriting-compliance.md` included verbatim for the product/claim mentioned (not required on product copy, PNs, or IAMs)
- [ ] Ends with a clear next step
